Eindreflectie

Ik vind dat ik mijn doelen heb bereik en meer. Ik ging naar PXL voor meer te leren over programmeren en wat bedrijfservaring op doen. Dit werd vervult door de lessen en uiteindelijk mijn stage. Maar ik kreeg ook veel meer zoals bijvoorbeeld softskills. Ik ben beter geworden met planning en inschattingen vermogen. Ik ben ook meer zelf zeker doordat ik meer ervaring heb. Werken in een groep lukt ook nu beter op een gestructureerde manier. Dankzij PXL zijn evenementen, heb ik ook meer connecties met mensen. Niet alleen met bedrijven maar ook met student. Ik kan altijd bij hun om advies vragen zelf nadat ik afgestudeerd ben. Het plannen van een project heb ik ook veel over geleerd. Ik had op het begin nooit verwacht dat dit een heel belangrijk deel is van het maken van een applicatie want ik altijd alleen kleine applicaties gemaakt met maximum 2 andere personen. Mijn sterkste kant was altijd mijn logisch redeneren. Dit is uiteindelijk veel beter geworden dan eerst want PXL heeft mij vaak gedreven om verder te gaan dan wat ik eerst kon. Met de kennis dat ik heb opgehaald bij PXL kan ik veel makkelijker nieuwe dingen leren dus ik ben van plan om mijzelf veel meer te verbeteren om zo de best versie van mijzelf te worden. Mijn plan is om te beginnen met werken zodat ik meer ervaring kan op doen want ik geloof dat je in de praktijk het meeste leert. Ik wil ook meer programmeer talen leren zoals C++ of Rust zodat ik kan deelnemen aan meerdere projecten. Kennis buiten programmeren wil ik ook uitbreiden zoals bijvoorbeeld een server hosten, netwerking, teammanagement. Mijn taal is mijn zwakste kan en hier zal ik altijd mijn best voor blijven doen zodat ik veel beter word met communiceren met andere personen. In mijn mening heb ik het meeste van multi- & disciplinariteit. Ik probeer altijd mij kennis zo veel mogelijk te verbreden ook al is het niet nodig voor mijn dagelijks leven. Ik ben ook goed geworden in mijzelf te disciplineren voor dingen te doen dat ik niet graag doe zoals teksten schrijven of presentaties geven. Empassie hoort ook wel bij mij. Ik ben een persoon dat als hij een doel heeft dan zal ik mij best doen om dit te bereiken. Dankzij dit kan ik mijzelf veel verder drijven dan normaal. Ik luister ook altijd naar andere mensen hun mening zodat wij altijd op een succesvol plan komen. Ik weet dat ik niet altijd het meeste weet over iets dus ik vind het belangrijk dat discussie mogelijk ik zelfs al ben ik eerst compleet tegen. Dit merkte ik het meest in het IT-project van het eerste semester van het derde jaar. Hier ben ik vaak in discussie gegaan met mensen en ze hebben mij vaak kunnen overtuigen voor iets op hun manier te doen ook als vond ik het op het begin een slecht idee.
